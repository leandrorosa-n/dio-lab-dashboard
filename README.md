# DIO | Projeto Dashboard de Vendas (Engenharia de dados)

Este projeto consiste no desenvolvimento de um dashboard de vendas em Excel, com foco em analisar dados relevantes na comercialização de veículos. Foram utilizados recursos como tabelas dinâmicas, gráficos interativos e segmentações de dados para facilitar a visualização das informações. O dashboard apresenta indicadores-chave, como faturamento, volume de vendas, produtos mais vendidos. Além disso, permite filtros dinâmicos para análise detalhada por região. O objetivo é transformar dados brutos em insights relevantes, proporcionando agilidade e precisão no acompanhamento dos resultados comerciais, sendo uma solução prática e eficiente para gestão de vendas.
